ELEMENTOS PARA ELABORAR ALGORITMOS

1. Variables. Recipientes especiales que usan para depositar datos 2. Control de entradas y salidas. Indicaciones para permitir que los datos ﬂuyan hacia el interior o exterior de la caja negra que efectúa el proceso. 3. Operaciones. La operación de asignación consiste en colocar un dato en el interior de una variable, mientras que las operaciones matemáticas son procesos elementales que transforman datos numéricos 4. Comparaciones y selecciones. Las comparaciones relacionan dos datos y sirven para imponer condiciones. Con ellas es posible tomar una decisión trasveriﬁcar si la condición resultó verdadera o falsa.
